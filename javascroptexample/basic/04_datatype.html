변수 선언 방식

        var : 변수 선언 키워드
        - 변수명 중복 가능(덮어쓰기 O)
        - 함수 레벨 스코프(function scope)
        - 재선언, 재활당 모두 가능
        - 호이스팅 시 underfined로 초기화됨 ※ ES6이후 권장 x

        let : 변수 선언 키워드(ES6)
        -변수명 중복 불가
        -블록 레벨 스코프(block scope)
        -재활당 가능 / 재선언 불가

        const: 상수 선언 키워드 (ES6)
        - 변수명 중복 불가
        - 블록 레벨 스코프
        - 재선언 , 재할당 모두 불가
        - 선언과 동시에 반드시 초기화 해야함
        - 객체/ 배열은 내부 값은 변경 가능 (불변 객체 아님)

        사용 권장 순서"
        1순위 : const (상수를 선언하거나, 변경되지 않아야 할 DOM 객체 참조 등)
        2순위 : let (변경 가능한 값을 다룰 떄만 사용)
        3순위 : var (가급적 사용 지양, 기존 코드 유지보수 외에는 거의 사용하지 않음)

자바스크립트의 자료형

        자바스크립트는 변수 선언 시 별도의 자료형을 지정하지 않음
        > 변수에 대입되는 값(리터럴)에 의해 자료형이 자동으로 결정됨

        [기본 자료형 (Primitive Type)]
        - String : 문자열
        - number :숫자 (정수와 실수 모두 포함)
        - boolean : 논리값 (true/flase)
        - undefined : 값이 할당되지 않은 변수
        - null : 값이 없음을 명시적으로 표현 (비어 있음)
        -bigint : (ES11 추가)아주 큰 정수를 표현할 때 사용

        [참조 자료형 (Reference Types)]
        -object : 객체
        [배열 (Array), 일반 객체 ({key: vlaue}), 날짜 (date), 정규식 (RegExp)등 포함]
        - function : 함수 (자바스크립트에서는 함수도 객체)

        ※ typeof 연산자를 사용하면 변수의 자료형을 확인 가능
